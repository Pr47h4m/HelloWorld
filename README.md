# Hello World

## Running C program
```bash
gcc helloworld.c
./a.out
```

## Running C++ program
```bash
g++ helloworld.cpp
./a.out
```

## Running Java program
```bash
java helloworld.java
```

## Running Javascript program
```bash
node helloworld.js
```

## Running Python program
```bash
python3 helloworld.py
```

## Running SH program
```bash
sh helloworld.sh
```

## Running Docker container
```bash
sudo docker run docker/whalesay cowsay "Hello World"
```
